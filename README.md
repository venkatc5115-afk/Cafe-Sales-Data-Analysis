Café Sales Data Analysis – Project Description (for GitHub)
📌 Project Title
Cafe Sales Data Analysis using Excel

📖 Project Overview
This project focuses on analyzing café sales data using Microsoft Excel to identify sales trends, customer purchasing patterns, and business performance insights. The analysis helps in understanding which products generate the highest revenue, peak sales periods, and overall café performance through interactive dashboards and visualizations.

The project demonstrates practical Excel skills used in real-world business analysis, including data cleaning, pivot tables, charts, formulas, and dashboard creation.

🎯 Objectives
Analyze overall café sales performance

Identify top-selling products and categories

Track monthly and daily sales trends

Calculate total revenue and average sales

Understand customer purchasing behavior

Create an interactive dashboard for decision-making

🛠️ Tools & Techniques Used
Microsoft Excel

Pivot Tables

Pivot Charts

Conditional Formatting

Data Cleaning

Excel Formulas (SUMIFS, IF, COUNTIFS, VLOOKUP/XLOOKUP)

Slicers & Filters

Dashboard Design

📊 Key Insights
Identified the highest revenue-generating products

Analyzed peak business hours and busiest days

Compared category-wise sales performance

Tracked monthly revenue growth

Found sales patterns to improve inventory and marketing decisions

📈 Dashboard Features
Sales by Product Category

Monthly Revenue Trend

Top Selling Items

Region/Branch-wise Performance (if applicable)

Interactive Filters and Slicers

KPI Cards for Total Sales, Profit, and Orders
